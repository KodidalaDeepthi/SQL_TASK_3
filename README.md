# SQL_TASK_3
Performed SQL data analysis on an E-commerce sales dataset using MySQL. Implemented queries with SELECT, WHERE, GROUP BY, ORDER BY, aggregate functions, subqueries, views, and indexes to extract meaningful business insights.

## 📌 Objective

The objective of this task is to use SQL queries to extract, manipulate, and analyze data from an E-commerce sales database. The project demonstrates fundamental SQL concepts such as filtering, sorting, grouping, aggregation, subqueries, views, and indexing.

---

## 📂 Dataset

**Dataset Name:** Ecommerce Sales Data 2024–2025

The dataset contains information about:
- Order ID
- Order Date
- Customer Name
- Region
- City
- Category
- Sub-Category
- Product Name
- Quantity
- Unit Price
- Discount
- Sales
- Profit
- Payment Mode

## 📈 Analysis Performed

- Displayed all records
- Filtered data using WHERE clause
- Sorted records using ORDER BY
- Calculated total sales and average profit
- Identified highest and lowest sales
- Counted total orders
- Analyzed sales by category
- Analyzed profit by region
- Analyzed sales by payment mode
- Used GROUP BY and HAVING clauses
- Created subqueries for advanced filtering
- Created a SQL View for summarized reports
- Created an Index to improve query performance
